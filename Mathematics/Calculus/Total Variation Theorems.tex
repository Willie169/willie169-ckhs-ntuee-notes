\sct{Total Variation Theorems}
\ssc{Implying or implied by bounded variation}
\sssc{Monotonicity}
If a real function is monotone on a closed interval, it is BV over it.

If a real function is monotone and bounded on an interval, it is BV over it.
\bpr
Obviously.
\epr
\sssc{Absolute continuity}
If a real function is absolutely continuous on a closed interval, it is BV over it.

If a real function is absolutely continuous and bounded on an interval, it is BV over it. 
\bpr
Suppose a real function $f$ is absolutely continuous on an interval $[a,b]$.

For every positive number $\varepsilon$, there exists a positive number $\delta$ such that for any finite sequence of disjoint subintervals $[x_k, y_k]$ of $[a,b]$,
\[\sum_{k}\left|y_{k}-x_{k}\right|<\delta\]
implies
\[\sum_{k}\abs{f(y_{k})-f(x_{k})}<\varepsilon.\]
Fix $\varepsilon=1$ and get obtain a $\delta$. Let $N$ be the smallest integer such that $N>\frac{b-a}{\delta}$. Let $x_k=a+\frac{k(b-a)}{N}$ and $I_k=[x_{k-1},x_k]$ for $k\in\bbZ\cap[0,N]$. Then
\[\sum_{j=1}^N\abs{f(x_k)-f(x_{k-1})}<1.\]
Take any fixed $I_k$. For any partition $P(t,m)$ of $I_k$, since $\sum_{j=1}^m(t_j-t_{j-1})<\delta$,
\[\sum_{j=1}^m\abs{f(t_j)-f(t_{j-1})}<1.\]
Thus
\[V_{I_k}(f)<1\]
Thus
\[V_a^b(f)<N<\infty.\]
\epr
\sssc{Differentiability}
A function of bouned variation on an interval is differentiable a.e. on it.
\bpr
Both functions in the Jordan decomposition is monotone and thus differentiable a.e.
\epr
\subsection{Total variation of differentiable function theorem}
\subsubsection{Total variation of differentiable real function}
If a real function $f$ is differentiable on $[a,b]$ and $f'$ is Riemann integrable over $[a,b]$, then the total variation of $f$ over $[a,b]$ is
\[V_a^b(f)=\int_a^b\abs{f'(x)}\dd{x}.\]
\bpr
Obviously.
\epr
